# Call Center Performance Analysis

## Overview
An end-to-end analysis of call center operations for a consumer electronics support team. 
This project examines agent performance, department efficiency, call resolution rates, and 
customer satisfaction across 1,772 calls recorded in January 2016.

## Tools Used
- Power BI (.pbix) — Interactive dashboard
- SSRS (.rdl) — Formal report
- Microsoft Excel / CSV — Raw data source

## Dataset
| Field | Details |
|---|---|
| Records | 1,772 calls |
| Period | January 2016 |
| Departments | Fridge, Air Conditioner, Toaster, Washing Machine, Television |
| Agents | Jim, Greg, Dan, Diane, Becky, Stewart, Martha, Joe |

## Key Metrics Analyzed
- Call answer rate (Y/N)
- Average speed of answer (seconds)
- Average talk duration
- Resolution rate per agent & department
- Customer satisfaction rating (1–5 scale)

## Key Findings
- Overall answer rate: 82% (1,455 of 1,772 calls answered)
- Average customer satisfaction: 3.45 / 5
- Departments tracked: 5 product categories

## Files
| File | Description |

1. Open `Call_Center_Performance_Analysis.pbix` in Power BI Desktop
